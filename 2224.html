Snipaste 如何使用快捷键切换贴图显示/隐藏状态

Snipaste帮助中心 / 2025年10月15日

Snipaste 中切换所有贴图的显示/隐藏状态，默认的快捷键是：

Shift+F3

详细操作说明：

  1. 功能描述： 这个快捷键是一个“全局切换”开关，按下一次会隐藏所有当前显示在桌面上的贴图窗口。再次按下，所有被隐藏的贴图会重新显示，回到它们之前的位置。
  2. 目的： 当您想要专注于工作，但又不想销毁（关闭）贴图时，此快捷键非常方便。被隐藏的贴图仍然存在于内存中，不会被计入“已关闭”的贴图数量。
  3. 其他隐藏贴图的方式：
    • 隐藏单个贴图： 在贴图上左键双击。
    • 关闭单个贴图： 按下 Esc 键，或按下 Ctrl + W。

如何自定义快捷键（如果 Shift + F3 不方便）：

如果 Shift + F3 与您常用的其他软件快捷键冲突，您可以进入 Snipaste 的设置进行更改。

  1. 进入首选项： 在系统托盘（任务栏右下角）找到 Snipaste 图标，右键点击 → 选择 “首选项…” (Preferences…)。
  2. 导航到控制选项卡： 在打开的窗口中，切换到 “控制” (Control) 选项卡。
  3. 查找对应设置： 在 “全局快捷键” (Global Hotkeys) 部分，找到名为 “隐藏/显示所有贴图” (Hide/Show all images) 的选项。
  4. 修改快捷键： 点击旁边的输入框，按下您想要设置的新快捷键组合（例如 Ctrl + Shift + H）。
  5. 保存并应用。

Snipaste 快捷键切换贴图显示/隐藏状态常见问题

Snipaste 中，“隐藏贴图”(Shift + F3) 和 “关闭贴图”(Ctrl + W 或 Esc) 在功能上有什么根本区别？

根本区别在于状态和可恢复性。

总结：

如果您确定不再需要某个贴图，请使用 Ctrl + W (关闭)。

如果您只是想暂时清空屏幕，请使用 Shift + F3 (隐藏)。

当我将贴图设置为“鼠标穿透”模式后，如何通过快捷键临时取消穿透状态，以便对贴图进行移动或操作？

“鼠标穿透”模式的切换需要您在 Snipaste 首选项中设置一个专门的全局快捷键。

再次按下该快捷键，即可切换回穿透状态。

设置快捷键：

进入 Snipaste 首选项 → “控制” → “全局快捷键”。

找到 “鼠标穿透开关” (Toggle mouse click-through) 选项，并为其设置一个容易记住的快捷键（例如 Ctrl + Shift + X）。

切换操作：

当贴图处于穿透模式（无法点击）时，按下您设置的快捷键 (Ctrl + Shift + X)。

如果您的鼠标正悬停在该贴图上，该贴图会立即退出穿透模式，您可以对其进行移动或右键操作。

贴图的快捷键操作能否与透明度设置结合，实现“临时显示，不操作时自动半透明”的效果？

Snipaste 自身没有“不操作时自动半透明”的规则，但你可以利用透明度调节快捷键和鼠标穿透功能来实现类似的高效工作流：

操作完成后，再次按下 “鼠标穿透开关” 快捷键，它将恢复到半透明且可穿透的状态。

设置透明度： 将贴图的透明度设置为你想要的半透明状态（默认是 Ctrl + 鼠标滚轮 或 Ctrl + + / - ）。

设置穿透： 启用该贴图的鼠标穿透模式（见 Q2），使其既透明又不会妨碍点击下方的窗口。

临时完全显示（恢复操作）：

当你需要看清或操作该贴图时，按下 “鼠标穿透开关” 快捷键。

退出穿透模式后，贴图将变为完全不透明（方便查看），并且可以被移动或编辑。
